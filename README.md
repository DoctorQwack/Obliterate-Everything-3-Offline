This is an offline version of Obliterate Everything 3 (OE3)

Current features:

-Store works
-Campaign is playable
-Local save files have been added
-all currancy is now free

To play
-download the zip from [releases]
-extract zip
-launch the file named "Launch OE3 Offline.bat
-wait for the local server to spin up
-the game will launch in your default browser
-after it loads click start
-type your desired username into the login page, no password or account creation required

Enjoy!
